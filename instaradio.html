Instaradio Navigation


First impressions and suggestions






Kevin from Instaradio was looking to improve their mobile experience. I walked through the android application and documented my feedback. I also created some wireframes to create a more consistent style and simplify the navigation.

First impressions

I signed up for Instaradio, jotted down my thoughts and took screenshots along the way. I paused at at each screen to think more about what I expected and why I reacted in certain ways. Here's some selected feedback I had.

Navigation suggestions

I also created some wireframes for a new navigation that is more aligned with the user's primary intentions.

Mobile experiences

Kevin said the feedback was great and a lot of it is spot on. Although I didn't work with them on further designs, I was glad to provide some insight.

By looking critically at each screen of the mobile application, I learned a lot more about designing mobile experiences. A key difference between web and mobile is the seamless transitions between pages, which is also becoming more popular in many web applications.
